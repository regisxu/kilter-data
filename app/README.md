# Kilterboard 攀岩记录查看器

一个基于 Web 技术的 Kilterboard 攀岩记录查看和统计分析工具。无需安装，直接在浏览器中打开即可使用。

## 功能特性

### 📋 记录列表
- 查看所有完攀（Ascent）和尝试（Bid）记录
- 按日期分组显示（今天/昨天/具体日期）
- 快速搜索线路名称
- 点击卡片查看详情

### 📊 统计分析
提供四个统计视图：

| Tab | 功能 |
|-----|------|
| **概览** | KPI卡片、完攀/尝试饼图、角度分布、趋势 |
| **难度分析** | 难度金字塔、完攀率柱状图、尝试次数分布 |
| **趋势分析** | 活动热力图、周/月/年趋势、星期偏好 |
| **SQL探索** | 自定义SQL查询、可视化结果、CSV导出 |

### 🔍 筛选功能
- 时间范围（最近7天/30天/90天/年份/自定义）
- 记录类型（完攀/尝试）
- 攀岩角度（0°-70°）
- 难度范围（滑块选择）
- 线路名称搜索

## 使用方法

### 1. 准备数据库文件
使用 `sync/kilter_sync.py` 同步数据到本地 SQLite 数据库（生成 `kilter.db` 文件）。

### 2. 打开应用
双击 `index.html` 或在浏览器中打开：

```bash
# 使用 Python 简单服务器（推荐）
cd app
python -m http.server 8080

# 然后访问 http://localhost:8080
```

### 3. 选择数据库
第一次打开时，点击"选择数据库文件"按钮，选择同步好的 `kilter.db` 文件。

### 4. 缓存机制
- 数据库文件会缓存到浏览器的 IndexedDB 中
- 下次打开自动加载，无需重复选择
- 点击 🔄 按钮可清除缓存并重新选择文件

## 技术栈

- **前端框架**: 原生 HTML5 + CSS3 + JavaScript
- **数据库引擎**: [sql.js](https://github.com/sql-js/sql.js) - WebAssembly SQLite
- **图表库**: [ECharts](https://echarts.apache.org/) - 百度开源可视化库
- **数据存储**: IndexedDB （浏览器本地存储）

应用采用响应式设计，完美支持手机浏览器访问。建议：

- 使用 Chrome/Safari/Edge 等现代浏览器
- 可添加到手机主屏底（PWA 模式）获得类似原生应用体验

## 注意事项

- 数据库文件不会上传到任何服务器，仅在本地处理
- 缓存数据保存在浏览器中，清除浏览器数据会丢失缓存
